# ParanoiaSkills Banner Text Effect

## Style Prompt

Premium dark technical command-center banner for an AI-assisted game design operating system. The text layer should feel like a precise interface element, not a poster slogan: luminous but restrained, readable at GitHub README scale, and integrated with the cyan/amber/violet data streams in the background.

## Colors

- Background ink: `#030912`
- Primary text: `#f5f8fc`
- Paranoia accent: `#31e1d6`
- Violet control glow: `#9a6cff`
- Amber evidence glow: `#f6b84a`
- Muted subtitle: `#dce8ee`

## Typography

- Primary: `Segoe UI`, `Inter`, system sans-serif
- Weight: heavy for title, regular for tagline

## Motion / Text Effect

- Title uses layered glow, subtle chromatic ghosts, gradient text fill, and a diagonal light sweep.
- Subtitle sits on a translucent precision bar with a thin scanline.
- Version badge uses a small cyan glass chip with soft glow.

## What Not To Do

- No dense explanatory copy in the image.
- No fake tiny UI text that becomes illegible.
- No mascots, characters, or decorative clutter over the title.
- No low-contrast text over bright interface panels.
- No large purple-blue gradient blanket; keep accents balanced with cyan and amber.
